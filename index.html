What is CSS

  1. CSS stand for cascading style sheet
  2. CSS describe how HTML elements areto be displayed on screen, paper, or in other media
  3. CSS is not necessaryCss saves a lot of workIt can control the multiple web pages all at once
  4. External Stylesheet are stored in CSS file

CSS can added to HTML documents in 3 ways:

  • Inline- by using the style attribute inside HTML elements
  • Internal - by using a style element in the head section
  • External- by using a link element to link to an external CSS file
  • The most common way to add CSS, is to keep the style in external CSS file. However in this tutorial we will use inline and internal styles,because this is easier to demonstrate, and easier for you to try it yourself

When comparing CSS class vs ID, the difference is that CSS class applies a style to multiple Elements. ID, on the other hand, applies a style to one unique element. ID is also special in that you can use special URL to link directly to an element and it's used by javascript

In CSS, selectors are used to target a specific element or range of element on web page. Once an element has been targated, a style or set of style can be applied to the element

There is wide range of selectors Avalaible. Two of the most commonly used are class and ID. Both are used to target elements to which a style should be applied
